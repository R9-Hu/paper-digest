    🆕 Today's Digest — Agentic AI / LLM Agents¶

    New papers/posts collected 2026-06-18.

    19 new today

    📋 In brief¶

    This batch clusters around two interlocking concerns: making agentic systems more capable and making them safer to deploy. On the capability side, several papers push on training efficiency and credit assignment for RL-trained agents — SC-GRPO, GraphPO, and EfficientRollout all attack the rollout bottleneck from different angles, while SGCD and Guava tackle the off-trajectory generalization problem for GUI and embodied manipulation agents. On the infrastructure and safety side, a parallel set of papers formalizes what current systems get wrong: TRAP exposes an impossibility result for prompt-level privacy defenses, GateMem finds no existing memory agent satisfies joint utility-access-forgetting requirements, and the concurrency-anomalies paper delivers the first machine-verified consistency hierarchy for multi-agent runtimes. Cutting across both concerns is a preoccupation with multi-agent coordination architecture — protocol taxonomies, meta-skill distillation (Skill-MAS), game-theoretic iterative best-response (MAFP), and learned orchestration of specialist models (SciOrch) — suggesting the field is actively converging on what the "right" abstraction layer for agent-to-agent interaction should look like. Rounding out the batch are several domain-grounded benchmarks (LegalWorld, WorldLines, TRAP, GateMem) that stress-test agents on long-horizon, stateful, and privacy-sensitive tasks where current systems still fall short.

    📝 Today's papers¶

    Code-Augur: Agentic Vulnerability Detection via Specification Inference¶

    Code-Augur introduces a security-specification-first paradigm for agentic vulnerability detection: instead of leaving an LLM agent's security judgments implicit, it forces the agent to commit its assumptions as executable in-source assertions (invariants), then uses a grey-box fuzzer to falsify those assertions in a reason-falsify-refine loop. On real-world benchmarks, it detects 34–370% more bugs than state-of-the-art agentic systems and discovered 22 previously unknown vulnerabilities, 16 of which have been fixed or confirmed.

    📌 Why selected: Abhik Roychoudhury (NUS); agentic LLM security via specification inference, watershed application

    A Technical Taxonomy of LLM Agent Communication Protocols¶

    This paper develops a five-dimension technical taxonomy to classify LLM agent communication protocols, applied to nine actively maintained open-source protocols. It identifies recurring architectural patterns and predicts short-term convergence toward unified agent-to-agent and agent-to-context protocols, with long-term evolution toward a federated, layered protocol stack.

    📌 Why selected: Alois Knoll (TU Munich); foundational taxonomy of agent communication protocols, high citation potential

    TRAP: Benchmark for Task-completion and Resistance to Active Privacy-extraction¶

    TRAP is a benchmark that jointly evaluates whether LLM agents can use private information to complete tool-calling tasks while resisting explicit extraction attacks. Evaluated across 22 models, all exhibit near-total privacy leakage at competitive task accuracy. A formal impossibility result shows no prompt-based defense can escape this trade-off; structural private field isolation (hash-key substitution) is proposed as a viable alternative.

    📌 Why selected: First benchmark explicitly testing agent task-completion vs. active privacy extraction resistance

    GateMem: Benchmarking Memory Governance in Multi-Principal Shared-Memory Agents¶

    GateMem is a benchmark that evaluates LLM memory agents on three joint governance dimensions—utility, access control, and active forgetting—in multi-principal shared environments (hospitals, offices, campuses, households). Existing benchmarks assume single-user private memory; GateMem exposes that no current method simultaneously achieves strong recall, robust unauthorized-disclosure prevention, and reliable post-deletion non-recovery. Long-context prompting scores highest on the composite metric but is token-expensive and still leaks in multiple configurations.

    📌 Why selected: Underexplored multi-principal shared-memory governance; critical for real-world enterprise agent deployment

    Skill-MAS: Evolving Meta-Skill for Automatic Multi-Agent Systems¶

    Skill-MAS proposes a third paradigm for automatic Multi-Agent System (MAS) generation that decouples experience retention from parametric updates by treating the Meta-agent's orchestration capability as an evolvable natural-language artifact called a Meta-Skill. Through iterative Multi-Trajectory Rollout and Selective Reflection, it distills systemic orchestration principles without fine-tuning the underlying LLM. The evolved Meta-Skills generalize across unseen tasks and different backbone LLMs.

    📌 Why selected: Evolving meta-skills for automatic MAS generation; addresses capability-vs-experience dilemma

    LegalWorld: A Life-Cycle Interactive Environment for Legal Agents¶

    LegalWorld is a life-cycle simulation environment for Chinese civil litigation that models a causally connected five-stage chain (consultation → drafting → trial → appeal → second-instance trial) grounded in 75,309 real paired judgments. Unlike prior legal-agent benchmarks that evaluate isolated subtasks from shared ground truth, LegalWorld propagates artifacts and state across stages so that early decisions constrain later ones. The companion benchmark LongJud-Bench evaluates lawyer agents across eight legal capabilities over the full litigation life cycle.

    📌 Why selected: Life-cycle legal agent environment; first to model cross-stage constraint propagation vs. isolated tasks

    SAGE: Stochastic Prompt Optimization via Agent-Guided Exploration¶

    SAGE formalizes automatic prompt optimization (APO) as noisy black-box search over a structured semantic landscape, then introduces a multi-agent pipeline where agents execute diagnostic code to identify root-cause error patterns before generating prompt edits. Evaluated on three benchmarks and deployed on a production mental-health chatbot, SAGE compounds individually-underpowered A/B cycles into a cumulative +29.4% relative gain in next-day retention.

    📌 Why selected: Agent-guided stochastic prompt optimization; treats APO as black-box search rather than gradient descent

    Learning from Own Solutions: Self-Conditioned Credit Assignment for Reinforcement Learning with Verifiable Rewards¶

    SC-GRPO improves token-level credit assignment in RLVR by conditioning the model on its own verified rollouts, computing per-token KL divergence between the conditioned and unconditioned distributions, and using that KL as a multiplicative weight on GRPO gradients—without any external teacher or privileged information. It improves Avg@8 by 8.1% over GRPO and 5.9% over DAPO across five benchmarks including multi-turn agentic tasks. The paper also proves that naively using verified trajectories as distillation targets produces infeasible weighted-average solutions.

    📌 Why selected: Token-level credit assignment beyond GRPO; addresses core RLVR training inefficiency

    EfficientRollout: System-Aware Self-Speculative Decoding for RL Rollouts¶

    EfficientRollout accelerates on-policy RL training by applying self-speculative decoding (SD) to rollout generation: a W4-quantized copy of the current target model serves as the drafter, a roofline-model-based toggle activates SD only in memory-bound regimes, and an adaptive draft-length controller tracks improving acceptance across training. Over an accelerated AR baseline (veRL + vLLM), it reduces rollout latency by up to 19.6% and end-to-end step time by up to 12.7% while preserving target-model distribution.

    📌 Why selected: Self-speculative decoding cuts dominant RL rollout latency bottleneck in post-training

    GraphPO: Graph-based Policy Optimization for Reasoning Models¶

    GraphPO represents RLVR rollouts as a directed acyclic graph (DAG) rather than independent chains or trees, merging semantically equivalent reasoning states into equivalence classes to eliminate redundant exploration and derive dense step-level rewards from sparse outcome signals. It introduces a dual-group advantage estimator that separately rewards reasoning correctness and path efficiency. Across three LLMs and five reasoning benchmarks plus four agentic search tasks, GraphPO consistently outperforms all chain- and tree-based baselines under equal token or trajectory budgets.

    📌 Why selected: Graph-based policy optimization challenges independent-sampling assumption in RLVR

    Skill-Guided Continuation Distillation for GUI Agents¶

    GUI agents trained via behavior cloning on expert trajectories lack supervision for the off-trajectory states they encounter during closed-loop execution. SGCD (Skill-Guided Continuation Distillation) addresses this by running the plain policy to reach realistic off-trajectory states, then using a skill-guided policy to produce verified successful continuations for training. Applied to three VLM backbones on OSWorld-Verified, SGCD lifts success rates from the low-30% range to over 50%.

    📌 Why selected: Closes distribution-shift gap in GUI agents during closed-loop execution

    WorldLines: Benchmarking and Modeling Long-Horizon Stateful Embodied Agents¶

    WorldLines is a benchmark for long-horizon stateful embodied household agents that requires agents to maintain persistent, partially observable world states across multi-day interactions. It also proposes ObsMem, an observer-grounded memory framework that separates event evidence, world state, epistemic belief, and commitments into typed views. Experiments show existing memory systems fail systematically on state-overwrite, temporal reasoning, and embodied planning tasks where state must persist.

    📌 Why selected: First benchmark targeting long-horizon stateful memory in real-home embodied agents

    Enhancing Decision-Making with Large Language Models through Multi-Agent Fictitious Play¶

    Existing LLM multi-agent systems handle execution complexity through divide-and-conquer but fail on tasks where stakeholder decisions are mutually dependent — a problem the authors term stance entanglement. Multi-Agent Fictitious Play (MAFP) addresses this by mapping each stakeholder to an agent that iteratively best-responds to the empirical mixture of opponents' historical language policies, converting recursive mutual anticipation into a flat, iterative process. MAFP achieves the highest tournament strength (0.533) and robustness (0.421) across 13 competitive game and negotiation scenarios.

    📌 Why selected: Tat-Seng Chua (NUS); game-theoretic fictitious play addresses MAS coordination limits

    Decoupling Search from Reasoning: A Vendor-Agnostic Grounding Architecture for LLM Agents¶

    Native search grounding in LLM APIs bundles retrieval policy, provider choice, and generation behavior behind an opaque boundary, making production systems hard to tune or port. This paper presents Decoupled Search Grounding (DSG), an MCP-compatible gateway that externalizes retrieval as a structured tool layer with explicit caching, provider routing, and fallback. DSG nearly matches native search accuracy on SimpleQA (86.1% vs. 87.7%) at 91% lower search cost and eliminates a failure mode called Search-Induced Verbosity.

    📌 Why selected: Vendor-agnostic search grounding decouples retrieval from generation in production agents

    Do as the Romans Do: Learning Universal Behaviors from Heterogeneous Agents¶

    GRID (General Reward Inference and Disentanglement) is a social learning method that decomposes per-agent reward functions into a shared general reward and individual specific rewards by training an architecture with embedding, emphasis, and reconstruction losses. The general reward captures universally competent behaviors (safety, task norms) across a heterogeneous demonstrator population, yielding a generalist agent that outperforms standard imitation learning baselines and serves as a stronger fine-tuning prior.

    📌 Why selected: Natasha Jaques (Google Brain/UW) on learning universal behaviors from heterogeneous agent populations

    Guava: An Effective and Universal Harness for Embodied Manipulation¶

    Guava is a harness framework for embodied robotic manipulation that systematically identifies three critical design principles—iterative ReAct-style loops, semantic action abstractions, and multimodal observations—and uses them to distill frontier VLM capabilities into a 4B open-source model trained on fewer than 2K simulation trajectories, achieving competitive real-world performance with zero-shot sim-to-real transfer.

    📌 Why selected: Universal tool-use harness for embodied manipulation as alternative to end-to-end VLA

    Verified Detection and Prevention of Concurrency Anomalies in Multi-Agent Large Language Model Systems¶

    This paper formalizes four concurrency anomalies specific to multi-agent LLM systems—stale-generation, phantom-tool, causal-cascade, and tool-effect reordering—as TLA+ predicates, constructs a mechanically-verified consistency hierarchy L0 ⊊ ··· ⊊ L4, and deploys three Rust runtimes verified in Verus to prevent them. It is the first machine-checked consistency hierarchy for multi-agent LLM runtimes, grounded in real deployed frameworks (LangGraph, AutoGen, ByteDance deer-flow).

    📌 Why selected: Formal verification of concurrency anomalies in shared-state multi-agent LLM systems

    SciOrch: Learning to Orchestrate Expert LLMs for Solving Frontier Multimodal Scientific Reasoning Tasks¶

    SciOrch trains a lightweight 8B VLM (Qwen3-VL-8B) to orchestrate frontier commercial LLMs (GPT, Gemini, Claude) for multimodal scientific reasoning by decomposing questions into sub-problems and routing each to the best-suited model. Training RL over expensive API calls is made tractable via a three-phase MCTS that amortizes API cost across shared-prefix tree branches and decouples gradient updates from API latency. On 240 frontier scientific questions, SciOrch achieves 56.66% accuracy, +3.74% over the best single model at less than half the API cost of self-consistency baselines.

    📌 Why selected: Orchestrating complementary expert LLMs to exceed individual frontier models on scientific reasoning

    Dr-DCI: Scaling Direct Corpus Interaction via Dynamic Workspace Expansion¶

    DR-DCI extends Direct Corpus Interaction (DCI)—terminal-command-based agentic search—to large corpora by exposing retrieval as an agent-callable pull action that incrementally materializes documents into a bounded local workspace. This separates corpus-level candidate discovery (retrieval) from workspace-level evidence investigation (DCI shell tools), recovering both precision and scalability. On BrowseComp-Plus, DR-DCI achieves 73.3% accuracy while cutting cost and wall time by ~2.5× versus raw DCI.

    📌 Why selected: Direct corpus interaction via dynamic workspace expansion; moves agents beyond retriever-mediated evidence
