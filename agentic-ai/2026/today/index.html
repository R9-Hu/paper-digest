    🆕 Today's Digest — Agentic AI / LLM Agents¶

    New papers/posts collected 2026-06-24.

    17 new today

    📋 In brief¶

    This batch is unified by a single pressure: agents hitting walls when tasks grow long, messy, or expensive, and the community's search for principled ways to push through them. Memory is the central battleground — papers probe whether agents actually build durable user models (MEMPROBE), benchmark the cost-structure tradeoffs across twelve memory architectures, propose hybrid text-plus-code representations (Metis), and diagnose the self-confirmation trap that poisons experience replay (EDV). A second strong thread is reliability under scale: SAFARI and SHERLOC both reframe debugging as structured hypothesis search rather than full-context loading, Bayesian control turns coding-agent orchestration into sequential hypothesis testing, and Nexus Sampling attacks the KV-cache budget that caps how long any of this can run. Underneath the engineering sits a quieter conceptual argument — the "Critique of Agent Model" formalizes what most of these systems implicitly admit: today's agents are scaffolded pipelines, not endogenously capable agents, and closing that gap is the real open problem the rest of the batch is circling.

    📝 Today's papers¶

    SAFARI: Scaling Long Horizon Agentic Fault Attribution via Active Investigation ICML 2026¶

    SAFARI is a tool-augmented diagnostic framework that replaces passive full-trajectory loading with an iterative, hypothesis-driven Active Investigation loop to localize faults in long-horizon multi-agent system traces. It decouples diagnostic accuracy from context window size by equipping an Investigator Agent with read and search tools plus a persistent Short-Term Memory (STM). On constrained budgets, SAFARI outperforms state-of-the-art by 19–20% and maintains 0.58 precision even when the target fault lies 5× beyond the model's native context limit.

    📌 Why selected: ICML 2026; fault attribution for long-horizon multi-agent failures beyond context window

    OpenThoughts-Agent: Data Recipes for Agentic Models¶

    OpenThoughts-Agent (OT-Agent) is a fully open data curation pipeline for training broadly capable agentic language models via supervised fine-tuning (SFT). The project runs 100+ controlled ablation experiments across all pipeline stages and assembles a 100K-example dataset that fine-tunes Qwen3-32B to 44.8% average accuracy across seven agentic benchmarks—3.9 pp above the strongest prior open-data baseline (Nemotron-Terminal-32B at 40.9%).

    📌 Why selected: Open data recipes for broadly capable agentic models; fills critical training-data gap

    Are We Ready For An Agent-Native Memory System?¶

    This paper conducts the first systematic, data-management-perspective evaluation of agent memory systems, decomposing them into four modules and benchmarking 12 systems across 11 datasets. The central finding is that no single architecture dominates: effectiveness depends on workload–structure alignment, and highly structured systems rarely justify their order-of-magnitude higher costs.

    📌 Why selected: Frames agent memory as full data-management system; foundational framing for harness builders

    Qwen-AgentWorld: Language World Models for General Agents¶

    Qwen-AgentWorld introduces the first general-purpose Language World Models (LWMs) — 35B-A3B and 397B-A17B MoE variants — that simulate agentic environments across 7 domains (MCP, Search, Terminal, SWE, Android, Web, OS) via long chain-of-thought next-state prediction. The models are trained through a three-stage CPT→SFT→RL pipeline on 10M+ interaction trajectories, then applied in two complementary roles: as a decoupled environment simulator for scalable/controllable agentic RL, and as a unified agent foundation model via world-model warm-up. Both paradigms yield measurable gains over baselines trained solely in real environments.

    📌 Why selected: Alibaba Qwen team; language world models as planning substrate for general agents

    World Models in Pieces: Structural Certification for General Agents ICML 2026¶

    General agents operating in complex "big-world" environments cannot achieve uniform performance guarantees across all tasks, a limitation this paper formalizes as an impossibility theorem. The authors introduce structural certification, a transition-local framework that maps bounded goal-conditioned performance on specific transitions to provable entry-wise alignment of the agent's internal world model. The main result yields a tight \(O(1/n) + O(\delta)\) error bound on the certified transition dynamics estimate.

    📌 Why selected: ICML 2026; structural certification showing inevitable agent specialization; challenges uniform guarantees

    Metis: Bridging Text and Code Memory for Self-Evolving Agents¶

    Metis is a self-evolving agent system that maintains a hierarchical dual-representation memory combining natural-language text entries (plans, facts, pitfalls) with selectively promoted callable code tools. The key insight, backed by a controlled ablation study, is that text memory and code memory exhibit complementary trade-offs in construction cost, execution efficiency, and transfer reliability—neither alone is sufficient. On AppWorld, Metis improves task accuracy by up to 20.6% over ReAct while cutting execution cost by up to 22.8%.

    📌 Why selected: Hybrid text+code memory for self-evolving agents; directly applicable to harness design

    Bayesian control for coding agents¶

    Coding agent orchestration is recast as cost-sensitive sequential hypothesis testing: a Bayesian controller maintains a posterior over candidate correctness and decides, at each step, whether to query a cheap critic, trigger refinement, invoke an expensive verifier, or stop. Evaluated across six LLM generators and nine benchmarks, Bayesian control yields higher expected utility than fixed-rule baselines when verification is costly and critics carry informative but imperfect signal. The resulting posterior belief also functions as a calibrated uncertainty score that outperforms token-probability baselines.

    📌 Why selected: Bayesian uncertainty-aware tool-use orchestration; principled alternative to fixed rules (Nakov et al.)

    SHERLOC: Structured Diagnostic Localization for Code Repair Agents¶

    SHERLOC is a training-free fault localization framework that pairs a reasoning LLM with a compact, structured tool suite to identify buggy code regions and emit five-field diagnostic findings (root cause, solution idea, dependencies, etc.) rather than bare file paths. On SWE-Bench Lite and Verified it reaches state-of-the-art file-level localization, and injecting its findings into downstream repair agents yields +5.95 pp resolve rate while cutting total tokens by 23.1%.

    📌 Why selected: NVIDIA author; structured diagnostic localization halves wasted budget in code repair agents

    Escaping the Self-Confirmation Trap: An Execute-Distill-Verify Paradigm for Agentic Experience Learning¶

    LLM agents that learn from their own trajectories fall into a "Self-Confirmation Trap": wrong-but-internally-consistent executions are mistakenly stored as valid experience and then retrieved to repeat the same errors. EDV (Execute-Distill-Verify) breaks this loop by decoupling execution, distillation, and validation across heterogeneous agents, admitting experiences into memory only after consensus-based multi-agent verification.

    ReMMD: Realistic Multilingual Multi-Image Agentic Verification for Multimodal Misinformation Detection¶

    ReMMD introduces a benchmark (ReMMDBench) and agentic verifier (ReMMD-Agent) for multimodal misinformation detection under realistic conditions: multilingual posts, multiple images, graded five-way veracity labels, and mixed visual provenance. ReMMD-Agent decomposes posts into atomic claims, manages a persistent cross-point evidence memory bank, and outperforms all baselines while cutting cost 17.5–79.9% versus prior agents. The core finding is that realistic misinformation detection is an evidence-management problem, not just a perception problem.

    MEMPROBE: Probing Long-Term Agent Memory via Hidden User-State Recovery¶

    MEMPROBE is a benchmark that reframes long-term agent memory evaluation from downstream task success to direct recovery of hidden user state. It exposes a critical gap: task completion nearly saturates even for a memoryless baseline (~99.9%), while category-balanced memory recovery (\(B\)) plateaus around 0.6 and degrades further under realistic top-\(k\) retrieval. This reveals that behavioral metrics are blind to whether any durable user model was actually formed.

    AGORA: An Archive-Grounded Benchmark for Agentic Workplace Document Reasoning¶

    AGORA is a benchmark for evaluating LLM agents on archive-grounded document reasoning: locating sparse evidence across large, messy workplace document collections (9,664 files, 372M tokens across 8 domains) that exceed any model's context window. The benchmark requires deliberate exploration, cross-file reconciliation of inconsistent terminology/units/conventions, and numeric computation. Even the strongest evaluated model (Gemini-3.1-Pro) reaches only 59.4% accuracy, confirming the task is far from solved.

    Critique of Agent Model¶

    This position/framework paper draws a formal distinction between agentic systems (LLM pipelines whose competence resides in external scaffolding—tools, workflows, prompts) and agentive systems (agents whose capabilities arise endogenously from internalized goal, identity, decision-making, self-regulation, and learning). It argues that virtually all current "AI agents" are agentic, not agentive, and proposes the Goal-Identity-Configurator (GIC) architecture as a concrete path toward genuine machine agency.

    📌 Why selected: Eric Xing (CMU); foundational critique of what constitutes agency in LLM systems

    Forget Without Compromise: Nexus Sampling for Streaming KV-Cache Eviction Under Fixed Budgets¶

    Nexus Sampling is a training-free KV-cache eviction method for streaming LLM inference that replaces the universal deterministic top-K selection with weighted reservoir sampling, paired with a novel Nexus score that surfaces "bridge tokens" via an iterative attention walk. At 80% eviction, it stays within ~1 point of dense attention on LongBench and outperforms all top-K baselines on retrieval-heavy tasks, with up to 10× smaller per-sequence cache memory.

    📌 Why selected: Nexus sampling for streaming KV-cache eviction; critical infra for long-context agentic inference

    Semantic Browsing: Controllable Diversity for Image Generation¶

    Semantic Browsing frames text-to-image diversity as construction of a hierarchical semantic tree rather than stochastic sampling. A four-agent VLM workflow (Context Analyst, Brainstormer, Decision Maker, Critic) iteratively expands each tree node into structured JSON scene descriptions, which are then rendered by a prompt-adherent T2I model. The result is a navigable gallery of 27 images (\(3^3\) leaf nodes) where every variation traces to an explicit, user-interpretable semantic decision.

    ChartWalker: Benchmarking the Cross-Chart RAG Task¶

    ChartWalker is a benchmark framework for Cross-Chart Retrieval-Augmented Generation (RAG) that constructs multi-hop QA instances using hierarchical knowledge graphs (KGs) built from chart corpora. It addresses the brittleness of prior key-point-extraction approaches by enforcing semantic coherence and controllable granularity during path sampling. The best evaluated model achieves only 64% correctness, revealing substantial headroom for improvement.

    LingxiDiagBench: A Multi-Agent Framework for Benchmarking LLMs in Chinese Psychiatric Consultation and Diagnosis¶

    LingxiDiagBench is a large-scale multi-agent benchmark for evaluating LLMs on Chinese psychiatric consultation and diagnosis, built on LingxiDiag-16K, a 16,000-case synthetic EMR-aligned dialogue dataset spanning 12 ICD-10 categories. It reveals a sharp performance cliff: LLMs reach 92.3% binary depression–anxiety accuracy but collapse to 28.5% on 12-way differential diagnosis, and dynamic (interactive) consultation consistently underperforms static inference.
