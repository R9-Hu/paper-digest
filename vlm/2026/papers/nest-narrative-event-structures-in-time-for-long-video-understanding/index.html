    NEST: Narrative Event Structures in Time for Long Video Understanding¶

    🕒 Published (v1): 2026-06-18 02:05 UTC · Source: Arxiv · link

    Why this paper was selected

    Narrative event structure benchmark exposes long video understanding gap beyond mere token-length scaling

    TL;DR¶

    NEST is a benchmark of 1,005 full-length movies (avg. ~98 min) annotated with ~102 multimodal narrative events per film, covering temporal ordering, hierarchical composition, and long-range causal dependencies. It exposes a sharp capability gap in current VLMs: grounded event discovery (ETD <8%, EL <6%, EAE <11%) remains largely unsolved, while reasoning over a given event graph (ERE up to 44.42% F1 after fine-tuning) is more tractable. The dataset is unique in requiring models to connect events separated by an average of 32 minutes of video.

    Problem¶

    Existing long-video benchmarks either cap video length well below feature-film duration, rely on multiple-choice formats susceptible to selection bias, or evaluate needle-in-a-haystack retrieval rather than structured narrative understanding. No prior benchmark jointly tests event trigger detection, temporal localization, argument extraction, and causal/hierarchical relation extraction across hours-long narratives where events depend on temporally distant context (e.g., a job loss at 32:45 causing a breakup at 55:00). Atomic event datasets (VidSitu: avg. 10 s; VidEvent: avg. 82 s) miss hierarchical composition and long-range narrative causality.

    Method¶

    NEST is built via a semi-automatic annotation pipeline applied to 1,005 public-domain and licensed movies:

    1. Scene segmentation: PySceneDetect splits each film at contextual boundaries (~160 scenes/movie).
    2. Event trigger detection: OmniEvent (open-domain) augmented with LLM-constrained extraction over a PropBank predicate vocabulary.
    3. Event argument extraction: LLM-based semantic role labeling supplemented with GLEN and OmniEvent, following PropBank core roles (ARG0, ARG1) and modifier roles (ARGM-LOC, ARGM-TMP).
    4. Event relation extraction: LLM-based extraction of temporal (before/after/overlap), causal, preconditioned, hierarchical, and coreference relations.
    5. Temporal grounding: State-of-the-art temporal video grounding models and Gemini 2.5 Pro; scene boundaries used as conservative fallback.
    6. Coreference resolution: Maverick + LLM assigns canonical global IDs to entities across the film.

    Audio description (AD) tracks—professional narrations written for visually impaired viewers—serve as the primary event source, providing visually faithful textual grounding. A SILVER set (1,005 movies, LLM-assisted, ~100B tokens consumed) and a dense GOLD set (5 movies, 5 human annotators at \(15/hr, ~\)600 total) are released. GOLD-SILVER weighted Cohen's \(\kappa \approx 0.50\) versus inter-annotator \(\kappa \approx 0.57\).

    Four benchmark tasks are defined: ETD (identify trigger verb+context given video \(V\) and scene boundaries \([t_s, t_e]\)), EL (predict \([\hat{t}_s, \hat{t}_e]\) given event description, ~160 candidate scenes), EAE (fill argument slots given trigger and roles), ERE (classify \(r(e_i, e_j) \in \{\text{temporal, causal, preconditioned, hierarchical, coreference, no\_relation}\}\)). Fine-tuning uses Qwen3-Omni-30B with LoRA on 8 × H200 GPUs at 0.1 FPS with Flash Attention and gradient checkpointing.

    Key Contributions¶

    • NEST dataset: 1,005 full-length movies (1,639 total hours, avg. 97.87 min), 102 events/video, 5,100 relation pairs/video, avg. 32-min inter-event temporal distance—orders of magnitude beyond VidSitu (3.2 s) and VidEvent (6.2 s).
    • Multi-task framework: Four structured extraction tasks (ETD, EL, EAE, ERE) evaluated beyond multiple-choice, including open-ended and F1-based metrics with an LLM judge for semantic equivalence.
    • GOLD benchmark: Densely human-annotated eval set for 5 complete movies with inter-annotator agreement metrics.
    • Baselines: Comprehensive zero-shot and fine-tuned evaluation of 12 state-of-the-art long-video VLMs.
    • Data release: Pre-extracted video (SigLIP, CLIP, DINOv2) and audio (wav2vec2, CLAP) features, fine-tuned checkpoints, and a public-domain video subset.

    Results¶

    Zero-shot performance (best model per task): - ETD: 7.98% accuracy (LLaVA-Video 7B); fine-tuned Qwen3-Omni reaches 6.09% (frame sampling at 0.1 FPS degrades discovery) - EL: 5.89% accuracy (Qwen3-VL 30B); random baseline ~0.6% (~160 scenes), so best is ~9× chance - EAE: 10.62% accuracy (OVIS2.5 9B); fine-tuned Qwen3-Omni: 10.5% - ERE overall F1: 35.45% zero-shot (Qwen2.5-VL 32B); fine-tuned Qwen3-Omni: 44.42% - ERE by relation type (fine-tuned): coreference 100%, hierarchical 76.82%, preconditioned 38.26%, causal 62.12%, temporal 2.44% - Text-only ERE (GPT-5): 40.80% F1; Qwen2.5-VL 32B text-only: 35.45% — comparable to video-based, indicating visual grounding is not yet being exploited - Flashback detection (non-linear temporal subset) is separately evaluated as an ERE subtype; temporal F1 collapses to 2.44% even after fine-tuning

    Limitations¶

    • Temporal localization quality: State-of-the-art grounding models fail on multi-minute clips; scene boundaries used as fallback, making EL scene-level rather than frame-precise.
    • Sparse video sampling (0.1 FPS) forces models to miss visual cues between sampled frames, likely contributing to low ETD/EL/EAE scores.
    • GOLD annotation coverage: Only 5 movies (~70 events, ~50 relations each) are densely human-annotated; the bulk of the benchmark relies on SILVER (LLM-assisted) annotations.
    • Downstream tasks (long-video captioning, VQA) are described but not benchmarked in this work.
    • Audio modality underutilized: Only 3 of 102 event types are audio-specific; audio features are released but audio-grounded evaluation is limited.
    • Video access restrictions: Full videos not universally available; only public-domain subset released, limiting reproducibility for end-to-end visual models.

    Relevance to Vision-Language Models¶

    NEST directly stress-tests the narrative reasoning capability of long-context VLMs, revealing that extended context windows do not automatically confer narrative understanding—models that handle millions of tokens still score below 8% on event discovery from 98-minute films. The benchmark's structured extraction format (ETD/EL/EAE/ERE) provides granular diagnostic signal beyond aggregated multiple-choice accuracy, enabling the community to distinguish perceptual grounding failures from relational reasoning failures. For VLM researchers, the finding that text-only models (GPT-5 at 40.80% ERE F1) nearly match video-based models suggests that current video encoders are not effectively transmitting narrative-scale visual information to the language backbone. NEST thus sets a concrete, challenging target for next-generation long-video VLMs that must integrate visual, audio, and dialogue streams over hour-scale temporal spans.
