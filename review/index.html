    Week in review¶

    W25 collected 149 papers across four topics: VLM (40), Agentic AI (40), Harnesses (40), and Med-Foundation (29). VLM was dominated by Vision-Language-Action robotics work and a parallel stream on medical VLMs, while Agentic AI split between RL training-efficiency improvements and a growing safety/governance cluster. The Harness track showed the clearest conceptual maturation, with multiple papers now treating the harness architecture itself—not the underlying model—as the primary object of study. Med-Foundation was lighter in raw count but high-density on adaptation efficiency and clinical safety, with a notable absence of strong new pretraining proposals relative to prior weeks.

    Dominant themes¶

    • VLA scaling via unlabeled egocentric video. ACE-Ego-0, Motion-Focused Latent Action, and μ₀ all converge on mining human ego-video to extract latent action representations, bypassing expensive robot teleoperation data; this is now a consensus direction, not an outlier.
    • VLM grounding as the bottleneck for robotics and embodied AI. AffordanceVLA, APT, World Pilot, and the position paper each argue the limiting factor is not model capacity but the absence of principled physical-grounding mechanisms; embodied evaluation (ERQA-Plus, SC3-Eval) reinforces this by documenting exactly where grounding fails.
    • RL training efficiency for agent fine-tuning. SC-GRPO, GraphPO, EfficientRollout, STARE, RODS, and EnvRL all attack the rollout bottleneck from different angles (credit assignment, DAG deduplication, speculative decoding, entropy stabilization, dynamic data synthesis); this is the most concentrated research front of the week.
    • Harness architecture as a first-class engineering concern. LLM-as-Code Agentic Programming, the intent-execution gap paper, StaminaBench (6× gap from harness choice alone), APEX, and AgentArmor all argue that orchestration structure—not model capability—is the dominant performance variable; this framing is hardening into a field position.
    • Agent safety: privacy, over-privilege, and memory governance. TRAP (impossibility result for prompt-level privacy defenses), over-privileged tool selection, GateMem (no existing memory agent satisfies joint utility-access-forgetting), and NRT-Bench collectively define a safety triad that has no current solution.
    • PEFT and efficient adaptation for medical foundation models. PEFT-MedSAM, MoLRE, Deep Reprogramming Distillation, and Permutation-COMQ all reduce adaptation cost for large medical FMs, with clinical deployability (explainability, quantization) as a hard constraint.
    • Multi-agent coordination protocols. A taxonomy paper (five dimensions, nine protocols), Skill-MAS (meta-skill distillation), MATM (shared trajectory memory), and MAFP (game-theoretic iterative best-response) all probe the same open question: what is the right abstraction layer for agent-to-agent interaction?
    • Benchmarking blind spots exposed. NEST, FG-BMK, MedFM-Robust, UNICORN, AgentFairBench, and LabOSBench all document failure modes that aggregate metrics miss; benchmark construction is itself a research output this week, not just evaluation infrastructure.

    Gaps & blind spots¶

    • Audio/speech in multimodal agents is nearly absent. Only EChO-Agent and EventDrive touch non-visual sensing; there is no coverage of audio-VLM architectures, speech-grounded agents, or multimodal agents that integrate video+audio reasoning—a gap given how prominent these are becoming in production systems.
    • Retrieval-augmented VLMs are underrepresented. Only MODE-RAG appears; no coverage of RAG-specific retrieval strategies, dense passage retrieval for medical imaging, or how retrieval interacts with VLM hallucination at inference time.
    • Formal verification of harness properties is missing. The concurrency-anomalies paper is the only machine-verified result; no papers on formal specification, model checking, or verified composition of agent pipelines—despite GateMem and TRAP identifying precisely the properties one would want to verify.
    • Continual learning / catastrophic forgetting in agents and VLMs. The verbal RL retention-forgetting dilemma paper is the lone signal; no systematic coverage of how deployed agents adapt to non-stationary environments without forgetting—critical for long-lifecycle production systems.
    • Federated/privacy-preserving training for medical FMs. With 29 med-foundation cards, zero cover federated learning, differential privacy in training, or multi-site aggregation—a glaring gap given the clinical data sensitivity constraints these models face.
    • Agentic MCP/A2A protocol benchmarks are missing. The taxonomy paper classifies nine protocols but there is no empirical benchmark comparing them on latency, correctness, or interoperability; the protocol design space is taxonomized but not evaluated.
    • Inference efficiency for VLMs in production. Only SPARE (token pruning) and SPOT-E address serving-time efficiency; no coverage of speculative decoding for VLMs, KV-cache sharing across agent calls, or batched multi-modal inference—all relevant when running VLMs inside harnesses at scale.
    • Sim-to-real transfer for embodied VLAs. Multiple papers train on human ego-video or simulation but transfer analysis is thin; no papers specifically on domain randomization, photorealistic sim, or systematic sim2real gap quantification.

    Suggested next focus¶

    • Deep-read the RL training efficiency cluster (SC-GRPO, GraphPO, EfficientRollout, STARE, RODS). These are directly applicable to training your own agent harnesses on verifiable-reward tasks; the convergence on different solutions to the same rollout bottleneck suggests a synthesis or ablation comparison would yield actionable design choices.
    • Operationalize the harness failure taxonomy. The intent-execution gap paper, StaminaBench, and AgentArmor collectively define a failure vocabulary; map it against your own harness architecture as a structured audit—this is a one-time investment that pays forward into every subsequent harness build.
    • Track the TRAP + GateMem + over-privilege cluster as a safety spec. These three papers together define what a memory-safe, least-privilege agent harness would need to satisfy; treating them as a requirements document rather than separate papers is the right framing.
    • Commission a deeper read on VLA evaluation methodology (SC3-Eval, ERQA-Plus, Act2Answer). If you're building or evaluating VLA-adjacent systems, the closed-loop video evaluator pattern from SC3-Eval is immediately reusable and avoids expensive robot rollouts.
    • Add federated/privacy-preserving medical FM coverage. Zero papers this week on a topic that is a hard regulatory constraint for clinical deployment; add keywords and a targeted source search before the gap compounds.
    • Survey the multi-agent protocol landscape empirically. The taxonomy exists; what's missing is a benchmark. Watch for follow-on work or commission a targeted search on MCP/A2A/AgentProtocol benchmarks.

    Candidate new skills¶

    • harness-audit — Given a set of harness papers, extract a structured failure taxonomy (intent-execution gap, over-privilege, memory governance, underspecification) and map each failure mode to a concrete mitigation or design decision; the StaminaBench / intent-execution gap / GateMem / TRAP cluster makes this pattern recur enough to warrant codification.
    • vla-eval-extractor — Specialized digest reader for VLA/embodied papers that pulls out: embodiment type, training data source (teleoperation / ego-video / sim), evaluation protocol (open-loop / closed-loop / real-robot), and sim2real transfer method; the VLA thread is now large and recurring enough that a generic digest skill misses the field-specific axes that matter.
    • rl-training-comparator — Given a batch of RL-for-agents papers (SC-GRPO, GraphPO, EfficientRollout, STARE, etc.), produce a structured comparison table on: what bottleneck each targets, data requirements, wall-clock speedup claimed, and applicability to verifiable-reward settings; the rollout-efficiency cluster is now dense enough that per-paper digests obscure the comparative structure.
